# Data Analyst Portfolio

A collection of analytical projects demonstrating SQL proficiency, statistical analysis, data pipeline development, dashboard design, and business-oriented data storytelling.

## Projects

### [Project A: SQL Analytics Deep Dive](./project-a-sql-ecommerce/) · [View notebook on nbviewer](https://nbviewer.org/github/tobibolu/data-analyst-portfolio/blob/main/project-a-sql-ecommerce/analysis.ipynb)
Advanced SQL analysis of 100K+ e-commerce orders using DuckDB. Covers revenue breakdown with window functions, customer cohort retention, seller performance tiering, delivery funnel analysis, and LTV estimation. Every insight generated through SQL with business recommendations.

### [Project B: Nigerian Telecom Churn Pipeline](./project-b-fintech-churn/) · [View notebooks on nbviewer](https://nbviewer.org/github/tobibolu/data-analyst-portfolio/tree/main/project-b-fintech-churn/)
End-to-end churn analysis for MTN Nigeria telecom customers. Covers exploratory data analysis, feature engineering, A/B test simulation with statistical hypothesis testing (chi-squared, z-test, power analysis), logistic regression churn modeling with threshold optimization, and a Streamlit dashboard for monitoring at-risk customers.

### [Project C: Cohort & Retention Deep Dive](./project-c-cohort-retention/) · [View notebook on nbviewer](https://nbviewer.org/github/tobibolu/data-analyst-portfolio/blob/main/project-c-cohort-retention/analysis.ipynb)
Customer lifecycle analytics on the Olist e-commerce dataset. Covers cohort-based retention analysis, RFM segmentation (Recency/Frequency/Monetary scoring), 12-month LTV estimation by segment and geography, and churn risk flagging. Includes an interactive [Tableau Public dashboard](https://public.tableau.com/views/OlistE-CommerceCohortRetentionAnalytics/Story1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link) and a multi-page Streamlit app.

## Tools & Technologies

Python, SQL (DuckDB, PostgreSQL, BigQuery), Pandas, Plotly, Tableau Public, Streamlit, Scikit-learn, Jupyter
